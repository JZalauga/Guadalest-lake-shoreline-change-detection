# Guadalest-lake-shoreline-change-detection
Project for university usage.
Script to test 4 methods to detect lake shoreline change from 2020 to 2026. 

## Pipe
* Script to download Sentinel-2 imagery directly from the Copernicus API
* Preprocessing by croping and band alignment
* Shoreline detection and results comparison

## Methodology:
* NDWI threshold
* NDWI + NDVI threshold
* HSV threshold
* K-means Clustering
